# IPFS Archive

TEN.IO uses IPFS as decentralized storage infrastructure for audiovisual fragments, metadata and permanent references.

---

## Purpose

This archive connects:

- Audiovisual fragments
- Ethereum references
- ENS identity
- Public releases
- Gateway mirrors
- Persistent storage

---

## Infrastructure

- Network: IPFS
- Identity: ENS
- Archive: TEN.IO
- Gateway Provider: Pinata

---

## Structure

Each fragment may contain:

- CID
- Gateway URL
- Media type
- Release status
- Associated fragment
- External mirrors

---

Archive active.
