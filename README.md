# 🧱 TEN-IO

Audiovisual fragments archived through Ethereum, IPFS and ENS.

TEN.IO is a decentralized audiovisual archive where music, narrative, Ethereum infrastructure and permanent memory converge into fragment-based storytelling.

No farming.  
No promises.  
Just fragments of truth.
